# SEO Guidelines

## Objetivo

Criar uma landing page tecnicamente otimizada para mecanismos de busca, sem prejudicar conversão, legibilidade ou fidelidade ao produto.

## Requisitos mínimos

A página deve conter:

- Um único `h1`.
- Estrutura lógica de `h2` e `h3`.
- Title otimizado.
- Meta description.
- Conteúdo semântico.
- Textos alternativos em imagens, quando existirem.
- URLs internas relevantes.
- CTA visível.
- Conteúdo suficiente para explicar o produto.
- Boas práticas de performance.
- Layout responsivo.
- Marcação schema.org quando fizer sentido.

## Estratégia textual

Defina:

- Palavra-chave principal com base no produto real.
- Palavras-chave secundárias.
- Termos de intenção comercial.
- Termos de intenção informacional.
- Perguntas frequentes que possam capturar buscas long-tail.

## Restrições

Não pratique keyword stuffing.
Não gere conteúdo genérico.
Não crie promessas não comprovadas.
Não adicione páginas extras sem necessidade.
