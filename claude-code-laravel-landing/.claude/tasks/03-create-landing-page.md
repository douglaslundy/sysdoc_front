# Task 03 - Create Landing Page

## Objetivo

Substituir a página inicial atual do Laravel por uma landing page comercial, responsiva e otimizada para SEO.

## Passos

1. Identificar a rota `/`.
2. Identificar a view atual.
3. Criar backup lógico da estrutura anterior, se necessário.
4. Criar nova view da landing page.
5. Inserir copy de conversão.
6. Inserir estrutura SEO.
7. Criar seções de funcionalidades.
8. Criar seção de benefícios.
9. Criar seção “como funciona”.
10. Criar FAQ.
11. Criar CTA final.
12. Ajustar responsividade.
13. Validar build.

## Estrutura mínima da página

- Hero.
- Funcionalidades.
- Benefícios.
- Como funciona.
- Diferenciais.
- FAQ.
- CTA final.
- Rodapé.

## Critérios técnicos

- Um único `h1`.
- HTML semântico.
- Responsividade mobile.
- Assets funcionais.
- Nenhuma rota crítica quebrada.
- Sem funcionalidades inventadas.

## Critério de conclusão

A rota `/` deve renderizar a nova landing page com conteúdo fiel ao sistema.
