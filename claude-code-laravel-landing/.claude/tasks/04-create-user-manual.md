# Task 04 - Create User Manual

## Objetivo

Criar um manual de uso completo baseado nas funcionalidades reais do sistema.

## Passos

1. Usar `.claude/memory/feature-inventory.md` como fonte principal.
2. Criar introdução do sistema.
3. Explicar acesso e autenticação.
4. Explicar navegação.
5. Documentar cada módulo.
6. Criar passo a passo para cada funcionalidade.
7. Explicar permissões, se existirem.
8. Criar FAQ.
9. Criar solução de problemas.
10. Criar glossário.
11. Salvar versão intermediária em Markdown.

## Caminho sugerido

```txt
docs/manual-usuario.md
```

## Critério de conclusão

O manual deve permitir que um usuário final entenda e utilize o sistema sem depender do código.
