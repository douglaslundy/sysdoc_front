# Task 02 - Create Product Positioning

## Objetivo

Transformar o inventário de funcionalidades em posicionamento comercial.

## Passos

1. Identificar público-alvo.
2. Identificar problema principal resolvido.
3. Identificar benefícios centrais.
4. Identificar diferenciais reais.
5. Criar proposta de valor.
6. Criar mensagens principais.
7. Criar palavras-chave preliminares.
8. Registrar decisões em `.claude/memory/decisions.md`.

## Entregáveis

- Proposta de valor.
- Descrição curta do produto.
- Descrição longa do produto.
- Lista de benefícios.
- Lista de diferenciais.
- CTAs recomendados.
- Objeções e respostas.

## Critério de conclusão

A copy da landing page deve poder ser criada a partir deste posicionamento sem inventar informações.
