# Task 01 - Map Features

## Objetivo

Criar um inventário completo das funcionalidades reais do sistema.

## Passos

1. Mapear rotas públicas.
2. Mapear rotas autenticadas.
3. Mapear rotas administrativas.
4. Mapear CRUDs.
5. Mapear dashboards.
6. Mapear relatórios.
7. Mapear integrações externas.
8. Mapear notificações.
9. Mapear permissões.
10. Mapear fluxos principais.
11. Registrar evidências por arquivo.
12. Atualizar `.claude/memory/feature-inventory.md`.

## Regra de evidência

Cada funcionalidade deve conter pelo menos uma evidência, por exemplo:

- Rota.
- Controller.
- View.
- Model.
- Migration.
- Teste.
- Service.
- Job.
- Policy.

## Critério de conclusão

O inventário deve permitir escrever a landing page e o manual sem suposições.
