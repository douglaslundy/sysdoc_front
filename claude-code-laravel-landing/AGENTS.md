# AGENTS.md

## Finalidade

Este arquivo descreve os agentes especializados usados para executar o projeto. Cada agente possui responsabilidade clara e deve atuar apenas dentro do seu escopo.

## Agentes

1. `project-auditor`
   - Analisa estrutura, rotas, controllers, models, migrations, views, policies, jobs, services, seeders, testes e documentação.
   - Produz inventário técnico do sistema.

2. `laravel-frontend-engineer`
   - Implementa a nova landing page no Laravel.
   - Ajusta rotas, views, componentes, assets e responsividade.

3. `conversion-copywriter`
   - Cria copy comercial com foco em conversão.
   - Usa clareza, benefícios, prova, objeções, diferenciais e chamadas para ação.

4. `seo-specialist`
   - Define estrutura SEO da landing page.
   - Cuida de title, meta description, headings, conteúdo semântico, schema quando aplicável e performance textual.

5. `technical-documentation-writer`
   - Escreve o manual de uso do sistema.
   - Transforma funcionalidades técnicas em instruções claras para usuário final.

6. `pdf-html-manual-builder`
   - Gera versão HTML e PDF do manual.
   - Garante formatação, navegação, sumário e legibilidade.

7. `qa-reviewer`
   - Revisa consistência, funcionalidades, links, responsividade, SEO, erros técnicos e fidelidade ao projeto real.

## Regra central

Nenhum agente pode inventar informações. Toda entrega deve se basear em evidências encontradas no código ou em documentação existente no repositório.
